# UtilAndroid

O Generic é uma solução ORM para a plataforma de desenvolvimento android com SQLite.
Ele gera automaticamente códigos SQL para carregar e guardar os objetos.

# Gradle

```
dependencies {
    ...
    compile 'com.viana:androidutil:0.0.6'
}
```
